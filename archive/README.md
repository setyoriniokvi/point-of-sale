Archived files

This folder contains files moved out of the main project to keep the demo simple and focused.

Contents:
- `ui/` - old frontend pages (productmanagement.html, dashboard.html, cart.html)
- `backend-dockerfiles/` - per-service Dockerfiles previously used for microservice images
- `kubernetes/` - previous Kubernetes manifests for multi-service deployment

If you want to restore or reuse these files, copy them back into the project and adapt as needed.
